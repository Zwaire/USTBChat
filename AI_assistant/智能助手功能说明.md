# 智能助手文档

## 1. 基础说明

- 服务地址：`http://<AI服务IP>:5001`
- 请求方式：`POST`
- 数据格式：`application/json`
- 编码：`UTF-8`

说明：

1. 触发词统一为：`@智能助手`

------

## 2. 接口一：智能问答

### 地址

```http
POST /ai/reply
```

### 功能

当私聊或群聊消息中包含 `@智能助手` 时，服务器调用此接口获取 AI 回复。

------

## 3. 接口二：文件概要

### 地址

```http
POST /ai/summarize
```

### 功能

当用户上传文件后，并发送包含 `@智能助手` 的私聊或群聊消息时，服务器调用此接口获取文件概要。

------

## 4. 接口三：群聊氛围分析

### 地址

```http
POST /ai/atmosphere
```

### 功能

根据某个群最近几条消息判断群聊氛围，返回情绪和灯颜色。
 **只用于群聊。**

## 情绪

| emotion | label | color   |
| ------- | ----- | ------- |
| calm    | 平静  | #4287f5 |
| angry   | 愤怒  | #f54242 |
| happy   | 开心  | #f5d742 |
| sad     | 悲伤  | #888888 |
| afraid  | 害怕  | #a042f5 |
