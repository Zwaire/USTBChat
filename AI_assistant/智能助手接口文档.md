# 智能助手接口文档

# 1. 服务说明

服务地址：http://<我的IP>:5001

对外提供以下 3 个 HTTP 接口：

- `/ai/reply`：智能问答
- `/ai/summarize`：文件内容总结
- `/ai/atmosphere`：群聊氛围分析

### 基本信息

- 服务协议：HTTP
- 请求格式：`application/json`
- 返回格式：`application/json`
- 编码：UTF-8

------

# 2.统一返回格式

### 成功

```json
{
  "type": "接口类型",
  "status": 0,
  "...": "其他业务字段"
}
```

### 失败

```json
{
  "type": "接口类型",
  "status": 1,
  "error": "错误信息"
}
```

------

# 3. 智能问答接口

## 地址

```http
POST /ai/reply
```

## 功能

根据当前私聊或群聊消息生成简短中文回复。

该接口用于：

- 用户发送 `@智能助手 xxx`
- 或客户端点击某个 AI 触发按钮后，由服务器转发调用

## 请求示例

### 私聊示例

```json
{
  "type": "ai_reply",
  "username": "张三",
  "friendname": "李四",
  "message": "@智能助手 什么是局域网",
  "recent_messages": [
    {"username": "张三", "message": "你好"},
    {"username": "张三", "message": "@智能助手 什么是局域网"}
  ]
}
```

### 群聊示例

```json
{
  "type": "ai_reply",
  "username": "张三",
  "groupname": "group_1",
  "groupmessage": "@智能助手 什么是局域网",
  "recent_messages": [
    {"username": "李四", "message": "今天在做课设"},
    {"username": "王五", "message": "服务器还没调通"},
    {"username": "张三", "message": "@智能助手 什么是局域网"}
  ]
}
```

------

## 成功返回

```json
{
  "type": "ai_reply",
  "status": 0,
  "reply": "局域网是在一定范围内连接多台设备进行通信的网络。"
}
```

------

## 失败返回

```json
{
  "type": "ai_reply",
  "status": 1,
  "error": "不能同时传入message和groupmessage参数"
}
```

------

# 4. 文件内容总结接口

## 地址

```http
POST /ai/summarize
```

## 功能

对文件文本内容生成简短概要。

## 请求示例

### 私聊示例

```json
{
  "type": "ai_summary",
  "username": "张三",
  "friendname": "李四",
  "message": "@智能助手 帮我总结这个文件",
  "filename": "报告.txt",
  "content": "这里是文件提取后的文本内容……"
}
```

### 群聊示例

```json
{
  "type": "ai_summary",
  "username": "张三",
  "groupname": "group_1",
  "groupmessage": "@智能助手 帮我总结这个文件",
  "filename": "实验报告.txt",
  "content": "这里是文件提取后的文本内容……"
}
```

------

## 成功返回

```json
{
  "type": "ai_summary",
  "status": 0,
  "summary": "该文件主要介绍局域网聊天室系统的设计目标、模块划分、通信流程及智能助手相关功能。"
}
```

------

## 失败返回

```json
{
  "type": "ai_summary",
  "status": 1,
  "error": "content 不能为空"
}
```

------

# 5. 群聊氛围分析接口

## 地址

```http
POST /ai/atmosphere
```

## 功能

根据最近几条群聊消息分析当前群聊氛围，并返回：

- 情绪枚举值
- 中文标签
- 对应颜色

该接口**仅适用于群聊**。

## 请求示例

```json
{
  "type": "ai_atmosphere",
  "groupname": "group_1",
  "recent_messages": [
    {"username": "李四", "message": "终于跑通了"},
    {"username": "王五", "message": "太好了哈哈哈"},
    {"username": "张三", "message": "今晚可以睡觉了"}
  ]
}
```

------

## 成功返回

```json
{
  "type": "ai_atmosphere",
  "status": 0,
  "emotion": "happy",
  "label": "开心",
  "color": "#f5d742"
}
```

------

## 特殊说明

如果 `recent_messages` 为空，则默认返回平静状态：

```json
{
  "type": "ai_atmosphere",
  "status": 0,
  "emotion": "calm",
  "label": "平静",
  "color": "#4287f5"
}
```

------

## 失败返回示例

```json
{
  "type": "ai_atmosphere",
  "status": 1,
  "error": "atmosphere 仅支持群聊，必须传入groupname参数"
}
```

------

# 6. 情绪说明

`/ai/atmosphere` 返回的情绪值固定为以下 5 种之一：

| emotion | label | color     |
| ------- | ----- | --------- |
| calm    | 平静  | `#4287f5` |
| angry   | 愤怒  | `#f54242` |
| happy   | 开心  | `#f5d742` |
| sad     | 悲伤  | `#888888` |
| afraid  | 害怕  | `#a042f5` |
