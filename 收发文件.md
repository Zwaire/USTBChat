# 一、文件消息协议建议

统一一下消息格式，服务器按这个收：

## 私聊文件

```python
{
    "type": "file_message",
    "username": "张三",
    "friendname": "李四",
    "filename": "实验报告.txt",
    "content": "文件内容文本",
    "time": "2026-03-13 18:00:00"
}
```

## 群聊文件

```python
{
    "type": "group_file_message",
    "username": "张三",
    "groupname": "group_1",
    "filename": "实验报告.txt",
    "content": "文件内容文本",
    "time": "2026-03-13 18:00:00"
}
```

# 二、如果客户端字段名可能不统一，先加一个提取函数

把这几个方法加到 `ChatServer` 类里面。

```python
def extract_file_payload(self, msg: dict):
    """
    兼容几种常见文件字段写法
    返回: filename, content
    """
    filename = (
        msg.get("filename")
        or msg.get("file_name")
        or ""
    )

    content = (
        msg.get("content")
        or msg.get("file_content")
        or msg.get("text_content")
        or ""
    )

    file_obj = msg.get("file")
    if isinstance(file_obj, dict):
        if not filename:
            filename = file_obj.get("filename") or file_obj.get("file_name") or ""
        if not content:
            content = file_obj.get("content") or file_obj.get("file_content") or ""

    return str(filename).strip(), str(content)
```

# 三、私聊文件处理函数

继续加到 `ChatServer` 类里：

```python
def handle_private_file_message(self, msg: dict):
    """
    私聊文件转发
    """
    sender = msg.get("username")
    target = msg.get("friendname") or msg.get("target")
    filename, content = self.extract_file_payload(msg)

    if not sender or not target:
        return

    # 统一补全字段，避免客户端发来的字段不规范
    forward_msg = {
        "type": "file_message",
        "username": sender,
        "friendname": target,
        "filename": filename,
        "content": content,
        "time": msg.get("time")
    }

    # 转发给目标用户
    self.send_private(target, forward_msg)
```

# 四、群聊文件处理函数

也加到 `ChatServer` 类里：

```python
def handle_group_file_message(self, msg: dict):
    """
    群聊文件转发
    """
    sender = msg.get("username")
    groupname = msg.get("groupname") or msg.get("friendname")
    filename, content = self.extract_file_payload(msg)

    if not sender or not groupname:
        return

    forward_msg = {
        "type": "group_file_message",
        "username": sender,
        "groupname": groupname,
        "filename": filename,
        "content": content,
        "time": msg.get("time")
    }

    # 这里先沿用你们现有群广播逻辑
    # 如果你们有“只发群成员”的专用函数，优先替换成那个
    self.broadcast(forward_msg)
```
