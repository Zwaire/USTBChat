# 通信接口

## 客户->服务
```python
dict={
	"type":["register","login","add_friend","create_group","add_group","message","group_message", "request_pwd_find","seed", 
			"get_contacts_list","get_friend_list","get_group_list", ] 
	"username": "wzx" #str
	"code": "123"   #str
	"ip": "127.0.0.1" #str
	"friendname": "xjz" #str
	"groupname": "group_1" #str
	"message": "hello" #str
	"groupmessage": "hello" #str
}

```
- 信息部分键值可删减
  
* * *

### 以下给出部分示例
1. 注册信息
```python
dict={
	"type":"register",
	"username":"wzx",
	"code":"123",
	"ip":"127.0.0.1"
}
```
2. 添加好友信息
```python
dict={
	"type":"add_friend",
	"username":"wzx"
	"friendname":"xjz"
}
```
3.发送消息信息
```python
dict={
	"type":"message"
	"username":"wzx"
	"friendname":"xjz"
	"message":"hello"
}
```

## 服务->客户
```python
dict={
	"type":["register","login","add_friend","create_group","add_group","message","group_message"],
	"status":[0,1,2,3,4,5,6,7]  #int
	"seed": #str
}
```
***
### 给出部分示例
1. 注册成功
```python
dict={
	"type":"register"
	"status": 0
}
```
2. 注册失败
```python
dict={
	"type":"register"
	"status": 1
}
```
解释：
- 0：指执行成功包括添加好友，注册，登录等
- 1：指注册的用户名重复
- 2：指登录密码错误
- 3：添加的好友用户名不存在
- 4：群聊名已存在
- 5：成功创建群聊，但群主用户名不存在
- 6：想群聊添加用户时，该用户已经在群聊中
- 7：想向群聊添加用户时，对应群聊名不存在
- 8：想要找回密码时，用户名不存在
