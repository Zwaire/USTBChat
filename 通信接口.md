# 通信接口

## 客户->服务
```python
dict={
	"type":["register","login","add_friend","create_group","add_group","message","group_message"]
	"username": "wzx" #str
	"code": "123"   #str
	"ip": "127.0.0.1" #str
	"friendname": "xjz" #str
	"groupname": "group_1" #str
	"message": "hello" #str
	"groupmessage": "hello" #str
}
```
- 信息部分键值可删减
  
* * *

### 以下给出部分示例
1. 注册信息
```python
dict={
	"type":"register",
	"username":"wzx",
	"code":"123",
	"ip":"127.0.0.1"
}
```
2. 添加好友信息
```python
dict={
	"type":"add_friend",
	"username":"wzx"
	"friendname":"xjz"
}
```
3.发送消息信息
```python
dict={
	"type":"message"
	"username":"wzx"
	"friendname":"xjz"
	"message":"hello"
}
```
